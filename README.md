# Paschoalotto Test
Project for admission test at Paschoalotto Serviços Financeiros S/A

## Installation

Configure project

```
• Restore backup "paschoalotto-base" in PostgreSQL
• Configure the connection string in the file "appsettings.json"
• Run in project
```
